# AutoTask - Product Overview

AutoTask is an Android automation framework that enables automated task execution through accessibility services and UI automation. It provides a comprehensive suite of tools for automating app interactions, including:

- **Accessibility Service Integration**: Leverages Android's accessibility framework for reliable UI interaction
- **Task Automation**: Chain multiple actions (click, input, wait, find) into automated workflows
- **Image Analysis**: AI-powered image recognition using GLM API for intelligent UI element detection
- **Layout Inspection**: Hierarchical view of app UI structure for debugging and element identification
- **Floating Window Logging**: Real-time task execution logs displayed in a floating window overlay
- **Root Permission Support**: Optional root-based automation for enhanced capabilities
- **Screenshot Capture**: MediaProjection-based efficient screen capture for analysis

The framework is designed for developers and automation engineers who need to automate repetitive tasks across Android applications.
